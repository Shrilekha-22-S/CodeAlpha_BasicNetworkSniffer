# Basic Network Sniffer

## Overview
This project captures and analyzes network traffic packets using Python and Scapy.

## Features
- Packet Capture
- Source IP Detection
- Destination IP Detection
- Protocol Identification
- Payload Analysis
- Real-Time Monitoring

## Technologies Used
- Python
- Scapy

## How to Run

Install Dependencies

pip install scapy

Run

python network_sniffer.py
